# 📊 Sales Dashboard - Python

Dashboard interativo para análise de vendas desenvolvido em Python.

## Tecnologias

- Python
- Pandas
- Streamlit
- Matplotlib

## Funcionalidades

- Total de vendas
- Vendas por vendedor
- Vendas por ano
- Visualização interativa de dados

## Executar localmente

pip install -r requirements.txt

streamlit run dashboard.py
